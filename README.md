# C++ basic tutorials


## 

Code written based on the following tutorials:

    2 - Classes
    3 - Constructor/Destructor
    4 - Copy Constructor
    5 - Friend Function
    6 - Inline Function

    8 - Pointer in Classes
    9 - Static Functions
